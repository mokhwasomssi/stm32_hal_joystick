# stm32_hal_joystick

__2 JOYSTICK, ADC, STM32 HAL__

## 0. Development Environment
- MCU : STM32F411CEU6
- IDE : STM32CubeIDE
- Joystick Module X 2

## 1. Feature
- 3.3V input
- 12bits resolution
- Set joystick value : 0 - 1000


## 2. User Configuaration

- ADC1
    - IN0
    - IN1
    - IN2
    - IN3
